# react-server
some demo server for news portal

# how to
You will need to add `.env` file with fields:
```
PORT=
JWT_SECRET=
MONGO_HOST=
MONGO_PORT=
```

run scripts:  
1. clone repo
2. `yarn` / `npm i`
3. `yarn start` / `npm run start` - start server on `http://localhost:PORT`
4. `yarn dev` / `npm run dev` - start server in watch mode by nodemon.
